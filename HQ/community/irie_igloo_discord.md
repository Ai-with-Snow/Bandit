## Music Bot: FredBoat

### Installation
1. Visit https://fredboat.com/invite
2. Select Irie Igloo server
3. Authorize with Connect, Speak, Read/Send Messages

### Commands
| Action | Command |
|--------|---------|
| Play song | `;;play [song or URL]` |
| View queue | `;;queue` |
| Skip | `;;skip` |
| Stop | `;;stop` |
| Pause/Resume | `;;pause` / `;;resume` |
| Shuffle | `;;shuffle` |
| Help | `;;help` |

### Song Request Channel Pin
```
🎶 How to request music
• Join a voice channel (like 🎧 vibe-stream)
• Type ;;play [song name or YouTube link]
• Use ;;skip or ;;stop if you have the DJ role
• Keep lyrics & vibes mindful 🕊️
```
