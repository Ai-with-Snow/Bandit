## Grommet vs Eyelet — Quick Reference

| Feature | Grommet | Eyelet |
|---------|---------|--------|
| **Size** | Larger, heavier | Smaller, lighter |
| **Parts** | Front ring + back washer | Single piece with flared back |
| **Use** | Bags, banners, heavy fabric, production | Shoes, corsets, scrapbooking, light fabric |
| **Tool** | Bench press (heavy duty) | Pliers or light press |

---

## Recommended Tools

### Primary: Grommet Bench Press (Production Tool)
**For:** Bags, kits, merch, banners, clothing, heavy fabric

**Options:**
1. **Stimpson 405** — Industry standard (~$250-$300)
2. **GoldStar Heavy Duty Press** — Budget tabletop (~$80-$120)
3. **QWORK Hand Press Kit** — Entry-level with dies + grommets included

**Dies:** Interchangeable inserts for different sizes/shapes
- Round (universal, most common)
- Oval (fashion-forward, specialty)

### Secondary: Eyelet Pliers (Detail Tool)
**For:** Small accessories, straps, corsets, tags, decorative work

**Recommendation:** XOOL Upgrade (Oval + Round)
- Handles both die shapes
- Portable, affordable
- Good for detail work, not production volume

## Grommet Sizes

| Size | Inner Diameter | Best For |
|------|---------------|----------|
| #0 | 1/4" | Light fabric, tags |
| #2 | 3/8" | Standard bags, pouches |
| #4 | 1/2" | Banners, heavy canvas |
| #8-#12 | 3/4"+ | Industrial, tarps |
